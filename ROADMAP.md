# Invoixy - Product Roadmap 🚀

Welcome to the future of **Invoixy**! This roadmap outlines the planned features and milestones for the project. Since this is an open-source project, community feedback and contributions will heavily influence this direction.

## 🎯 Short-Term Goals (V1.1 - The Polish Update)
*Focus: Enhancing the user experience and adding frequently requested quality-of-life features.*

- [ ] **Dark Mode:** A sleek, fully integrated dark theme for late-night invoicing.
- [ ] **Custom Branding Upload:** Allow users to upload and manage their company logo directly from the Settings UI.
- [ ] **Custom Terms & Conditions:** Allow users to save default terms, conditions, and notes that automatically populate at the bottom of the invoice.
- [ ] **Flat Amount Discounts:** Currently, discounts are percentage-based. Add support for flat-amount discounts per line item.

## 🚀 Medium-Term Goals (V1.5 - The Power User Update)
*Focus: Expanding capabilities for businesses with more complex invoicing needs.*

- [ ] **Direct Emailing:** Integration with an email provider (like Resend or SendGrid) to email the PDF invoice directly to the client from the dashboard.
- [ ] **Payment Links:** Allow users to connect Razorpay or Stripe to embed clickable "Pay Now" links directly inside the generated PDF.
- [ ] **Multi-Currency Support:** Support for USD, EUR, etc., for businesses exporting services outside of India.
- [ ] **Recurring Invoices:** Templates and reminders for invoices that need to be generated monthly (e.g., retainers, subscriptions).

## 🌍 Long-Term Vision (V2.0 - The Enterprise Update)
*Focus: Turning Invoixy into a complete, multi-tenant billing powerhouse.*

- [ ] **Multi-Business Support (Tenancy):** Allow a single user to manage multiple different businesses/GSTINs from a single dashboard.
- [ ] **GSTR Export:** Generate and download the exact JSON formats required by the Indian Government's GST Offline Utility for easy tax filing.
- [ ] **E-Invoicing (IRN):** Direct integration with government portals to automatically generate e-Invoice IRN and QR codes.
- [ ] **Authentication & Cloud Sync:** Optional user accounts to sync data across multiple devices via a cloud database instead of local SQLite.
